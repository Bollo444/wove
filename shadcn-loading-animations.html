Usage Instructions

Spinner: Use the spinner icon from shadcn/ui icons.tsx with Tailwind's animate-spin class

Skeleton: Apply animate-pulse class to placeholder elements while content loads

Progress: Combine with shadcn/ui Progress component for determinate loading

Dots: Perfect for button loading states and inline loading indicators
